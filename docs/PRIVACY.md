# Markleaf Privacy Policy Draft

이 문서는 MVP 기준 개인정보 보호 정책 초안입니다.

## 핵심 원칙

Markleaf MVP는 오프라인 중심 앱입니다.  
사용자의 노트 데이터는 사용자가 직접 내보내거나 공유하지 않는 한 기기 밖으로 전송되지 않습니다.

## 수집하지 않는 정보

MVP는 다음 정보를 수집하지 않습니다.

- 이름
- 이메일
- 위치
- 연락처
- 계정 정보
- 노트 내용
- 태그
- 파일명
- 사용 통계
- 기기 식별자
- 광고 식별자
- 분석 이벤트

## 네트워크

MVP는 인터넷 권한을 사용하지 않습니다.

```text
android.permission.INTERNET
```

위 권한은 MVP에 포함하지 않습니다.

## 데이터 저장

노트는 사용자의 기기 내부 로컬 데이터베이스에 저장됩니다.

## 데이터 내보내기

사용자가 직접 실행한 경우에만 데이터가 앱 밖으로 나갈 수 있습니다.

- Markdown 파일로 내보내기
- Android 공유 시트로 공유
- 클립보드로 복사

## 향후 변경

향후 동기화, 백업, API, 로그인 기능이 추가될 경우 개인정보 정책을 별도로 재검토해야 합니다.
