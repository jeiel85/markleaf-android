핵심 기능

Markdown 중심

표준 Markdown 문법을 지원하여 어디서나 호환되는 노트를 작성하세요.

태그 기반 정리

복잡한 폴더 구조 대신 유연한 태그 시스템으로 노트를 분류합니다.

로컬 우선 (Privacy)

당신의 데이터는 오직 당신의 기기에만 저장됩니다. 인터넷 연결이 필요 없습니다.

빠르고 가볍게

불필요한 애니메이션과 무거운 기능을 배제하여 즉각적으로 반응합니다.

Markleaf의 철학

"조용하고 정돈된 나만의 기록 공간"

우리는 당신의 기록이 단순한 데이터 이상임을 알고 있습니다. Markleaf는 사용자의 데이터를 존중하며, 언제든 Plain Markdown으로 내보낼 수 있는 개방성을 지향합니다.
