# CHANGELOG

## v0.2.0 - 2026-04-30

### Added
- 실시간 마크다운 미리보기(Preview) 기능 추가 (에디터 상단 토글).
- 시스템 설정 연동 다크 모드(Dark Mode) 정식 지원.
- 브랜딩 가이드가 반영된 새로운 Sage Green 테마 적용.

- 에디터 내 마크다운 문법별 시각적 하이라이트 및 스타일링 개선.
- 전반적인 UI 색상 대비 조정을 통한 시독성 강화.


### Changed
- Integrated reusable agent operation templates into this repository's documentation structure.
