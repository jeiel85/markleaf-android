# Markleaf Branding Guide

## 목적

Markleaf는 특정 기존 메모 앱을 복제하지 않는 독립 브랜드다.  
이 문서는 브랜드, 이름, 색상, 문구, UI 방향이 기존 앱과 혼동되지 않도록 하기 위한 기준이다.

## 브랜드 핵심

Markleaf의 인상:

- 가볍다
- 빠르다
- 조용하다
- 정돈되어 있다
- 글쓰기에 집중한다
- 사용자의 데이터를 존중한다
- 개발자 친화적이고 오픈소스 친화적이다

## 이름

코드명 및 초기 앱 이름:

```text
Markleaf
```

의미:

- Markdown
- Mark
- Leaf
- 가볍게 쌓이는 노트
- 태그 기반으로 분기되는 생각

## 금지 사항

다음은 사용하지 않는다.

- Bear와 유사한 앱 이름
- Bear와 유사한 아이콘
- 곰 캐릭터 중심 정체성
- 기존 메모 앱과 거의 동일한 색상 조합
- 기존 메모 앱과 혼동될 수 있는 화면 구성
- 다음 문구:
  - Bear for Android
  - Bear clone
  - Android Bear
  - Alternative client for Bear
  - unofficial Bear client

## 안전한 포지셔닝 문구

영문:

```text
Lightweight Markdown notes for Android
Local-first Markdown note app
Tag-based writing and note organization
Portable notes in plain Markdown
```

한국어:

```text
Android용 가벼운 Markdown 메모 앱
로컬 우선 Markdown 노트
태그 기반 글쓰기와 메모 정리
Markdown 파일로 보관 가능한 개인 메모 앱
```

## 디자인 방향

- 텍스트 중심
- 충분한 여백
- 부드러운 타이포그래피
- 빠른 진입
- 과한 카드 UI 지양
- 과한 애니메이션 지양
- Material 3 기반이지만 기본 템플릿처럼 보이지 않게 조정

## 아이콘 방향

권장 컨셉:

- 잎
- 종이
- 마크다운 기호
- 얇은 선
- 간결한 형태

피해야 할 컨셉:

- 곰
- 발바닥
- Bear와 비슷한 동물형 캐릭터
- 특정 기존 메모 앱을 떠올리게 하는 강한 색상 조합

## 제품 톤

Markleaf의 문구는 짧고 명확해야 한다.

좋은 예:

```text
새 노트
검색
태그
휴지통
Markdown으로 내보내기
기기에만 저장됩니다
```

피해야 할 예:

```text
마법처럼 정리해드립니다
AI가 완성해드립니다
클라우드로 안전하게 보호됩니다
```

MVP에서는 클라우드나 AI 기능이 없으므로 관련 표현을 쓰지 않는다.
