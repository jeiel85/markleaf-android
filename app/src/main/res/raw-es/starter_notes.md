# Bienvenido a Markleaf

Markleaf es una app local-first de notas Markdown para Android. Esta pensada para abrir rapido, escribir de inmediato y mantener tus notas portables como archivos Markdown.

## Prueba primero

- Crea una nota nueva y captura una idea
- Agrega etiquetas como #idea dentro del cuerpo
- Conecta notas con `[[Escribe mejor con Markdown]]`
- Crea una copia ZIP desde Ajustes

Puedes editar o borrar estas notas iniciales cuando quieras. #inicio #guia

# Organiza con etiquetas y enlaces

Markleaf se organiza con etiquetas y enlaces en lugar de carpetas.

## Etiquetas

Escribe etiquetas como #proyecto, #reunion o #lectura en el cuerpo para agrupar notas relacionadas. Una nota puede tener varias etiquetas.

## Enlaces wiki

Envuelve el titulo de otra nota como `[[Copia y exportacion local-first]]` para conectar notas.

Esta nota tambien enlaza con `[[Bienvenido a Markleaf]]`. #organizar #enlaces

# Copia y exportacion local-first

En el MVP, Markleaf guarda tus notas en el dispositivo sin cuentas, analiticas, anuncios ni integracion con servidores.

## Lleva tus datos contigo

- Comparte una nota como Markdown.
- Exporta todas las notas como archivos Markdown.
- Crea y restaura copias ZIP desde Ajustes.

Por defecto, tus notas no salen del dispositivo hasta que las exportas, compartes o respaldas explicitamente. #copia #privacidad
